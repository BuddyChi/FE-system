##Symbol

```
var sym1 = Symbol();  // Symbol() 
var sym2 = Symbol('Buddy');  // Symbol(ConardLi)
var sym3 = Symbol('Buddy');  // Symbol(ConardLi)
var sym4 = Symbol({name:'ConardLi'}); // Symbol([object Object])
console.log(sym2 === sym3);  // false
```
